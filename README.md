# Portfolio Chat Interface

A responsive and interactive AI chat interface showcasing Nazarena's portfolio.

## Font Setup
For the custom font to work:
1. Make sure the `font/TT Wellingtons Trial Medium.otf` file is placed in the font folder
2. If you don't have the font file, replace it with another font or adjust the CSS accordingly

## Features
- Toggle between chat and text views
- Interactive AI responses
- Password-protected portfolio showcase
- Responsive design for all devices
- Custom typography

## Usage
Simply open the `index.html` or `ai-chat-interface.html` file in your browser to start using the interface.
